# git content

## `git clone`
克隆整个仓库
```git
git clone https://gitee.com/junbao-shell/software-config.git
```

只克隆指定的分支
```shell
git clone --single-branch --branch <branch_name> <remote=repository-url>

# 或者
git clone --single-branch -b <branch_name> <remote-repository-url>
```
